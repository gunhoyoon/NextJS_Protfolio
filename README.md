# 노션 나만의 포트폴리오

> 진행한 프로젝트들을 노션 포트폴리오에 담아 배포하였습니다.

# 사용 기술

Next js와 typescript를 통해 페이지를 만들었습니다.

로티 에니메이션을 사용해 UI를 구성하였습니다.

노션 api를 포스트맨으로 받아와 데이터 모델링을 하였습니다.

vercel을 이용해 배포하였습니다.

https://i18nexus.com/ 을 이용해 i18n국제화를 적용하였습니다.

## 프로젝트 실행 방법

> 터미널 Next portfolio에서

> npm install

> npm run dev
